## Project Title: AD 312 Building a Global Theme/Dark Mode Switcher
#### By: Amde Wubshet

### Breif Description
Because "Dark Mode" is a standard feature in most web applications, I learned how to build a Global Theme Switcher using the React Context API to manage a “Light” vs “Dark” theme across the entire application.

### Commands 
After installing the repo, navigate into your project directory: *cd theme-switcher-app*. Then start the development server: *npm run dev*.
